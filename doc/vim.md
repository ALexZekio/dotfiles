## 常用操作总结
### 快捷键
|    快捷键        |        功能        |
| ---------------- | ------------------ |
| ^u^              | 撤销操作           |
| ^Ctrl^+^R^       | 撤销重做           |
| ^J^              | 当前行与下一行拼接 |

### 移动篇
|    快捷键        |        功能        |
| ---------------- | ------------------ |
| ^h^              | 光标向左           |
| ^j^              | 光标向下           |
| ^k^              | 光标向上           |
| ^l^              | 光标向右           |
| ^%^              | 结对符号跳         |
| ^[[^             | 上一个`{`          |
| ^]]^             | 下一个`{`          |
| ^[]^             | 下一个`}`          |
| ^][^             | 上一个`}`          |
| ^{^              | 上一段落开头       |
| ^}^              | 下一段落开头       |
| ^Ctrl^+^]^       | 跳转关键字定义处   |
| ^Ctrl^+^o^       | 跳转回原位置       |

### 编辑篇
|    快捷键        |        功能        |
| ---------------- | ------------------ |
| ^a^              | 光标后插入         |
| ^o^              | 光标下一行插入     |
| ^O^              | 光标上一行插入     |
